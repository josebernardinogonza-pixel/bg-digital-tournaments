# BG Digital Solutions | Social Tournament Platform

## Project Overview
This repository contains the frontend and automated infrastructure for **BG Digital Solutions'** flagship social tournament platform. We specialize in high-engagement, mobile-first experiences designed for the Latin American iGaming market.

## Technical Stack
* **Frontend:** HTML5, Tailwind CSS, and JavaScript (ES6+).
* **Architecture:** Mobile-first responsive design, optimized for high-speed performance on Android and iOS.
* **Backend Integration:** Designed for REST API consumption (iGaming Aggregators).
* **CI/CD:** Automated workflows via **GitHub Actions** for leaderboard updates and scheduled event management.
* **Database:** Real-time data synchronization for global rankings.

## Key Features
* **Dynamic Leaderboards:** Live ranking updates based on tournament performance.
* **Modular API Structure:** Ready for seamless integration with major game providers (Pragmatic Play, SoftSwiss, etc.).
* **Engagement Tools:** Built-in countdown timers and automated event triggers.

## Market Focus
Currently targeting the **Mexican (LATAM)** market, focusing on user acquisition through gamification and high-fidelity mobile experiences.
